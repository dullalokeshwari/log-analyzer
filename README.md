# log-analyzer
Python file handling & automation project for analyzing log files and reports automatically
# Log File Analyzer

## Objective
This project reads a log file, counts INFO, WARNING, and ERROR messages, and generates a summary report automatically.

## Features
- Reads log files
- Counts log message types
- Displays results on the screen
- Generates a report.txt file automatically

## Technologies
- Python 3
- File Handling

## Files
- log_analyzer.py
- sample_log.txt
- report.txt

## How to Run
python log_analyzer.py
